## 🚀 About the Project
I built this project to help beginners figure out exactly where to start their data journey. It breaks down the three foundational pillars of data analysis into digestible, easy-to-understand chapters complete with practical exercises.

### 📚 What the eBook Covers:
1. **Excel Basics:** Interface navigation, Data Cleaning (TRIM, Remove Duplicates), VLOOKUP/XLOOKUP, Pivot Tables, and Power Query automation.
2. **SQL Fundamentals:** Database basics, filtering data (SELECT, FROM, WHERE), aggregations (GROUP BY), and combining tables (INNER JOIN).
3. **Python (Pandas):** DataFrames vs. Series, handling missing data, and performing advanced data manipulation.

### 💻 Tech Stack (Landing Page)
The landing page was built entirely from scratch to ensure maximum performance and a premium aesthetic without the need for heavy frameworks.
* **HTML5:** Semantic structuring
* **CSS3:** Custom properties, Flexbox, glassmorphism effects, and dynamic hover animations.
* **Vanilla JavaScript:** DOM manipulation and parallax 3D hover effects.
* **Python:** Used a custom script to compile the markdown files into a portable, styled HTML eBook.
