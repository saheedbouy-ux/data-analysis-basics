# Design Document

## Project: Data Analysis Basics eBook Landing Page

### Core Aesthetics
- **Theme**: Premium Dark Mode
- **Vibe**: Tech-focused, modern, high-end, dynamic
- **Background**: `#0a0a0f` with ambient glowing orbs.
- **Typography**: `Inter` for body copy, `Outfit` for headings and logo.

### Color Palette
- Text Main: `#f0f0f5`
- Text Muted: `#9494a0`
- SQL Accent: `#3b82f6` (Blue)
- Python Accent: `#eab308` (Yellow)
- Excel Accent: `#10b981` (Green)

### Tech Stack
- HTML5 (Semantic Structure)
- Vanilla CSS3 (Custom properties, Flexbox/Grid, Animations)
- Vanilla JS (Interactivity, Parallax effects)

- Downloadable eBook: `ebook-draft.md`
